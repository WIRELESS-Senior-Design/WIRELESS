ECE Senior Design Group 22 - Spring-Summer 2026

W.I.R.E.L.E.S.S.

Wireless Intelligent Remote Environment Lighting & Electrical-appliance Smart System

WIRELESS is a communication system for devices using low-frequency RF communication. It implements a custom communication protocol that is able to control each device from a central hub, making use of a touchscreen-based User Interface (UI). Users are able to add, edit and delete devices with the UI.
